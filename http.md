HTTP es un protocolo de comunicación que define cómo los navegadores y servidores intercambian información en la WWW. Opera en la capa de aplicación del modelo de capa de red. Está basado en un modelo de request (cliente) y response (servidor), donde cada petición es independiente (sin estado). Permite la transferencia de diversos tipos de datos: HTML, imágenes, videos, etc. Si bien originalmente fue pensado para navegadores web, hoy día se usa en todo tipo de comunicación cliente-servidor.
